# 医药ETF易方达(512010) 量化策略分析

本项目通过获取医药ETF易方达（512010.SH）最近一年的交易数据，生成可视化HTML仪表板。

## 📊 项目内容

- **K线图**：日K线蜡烛图，含MA5/MA10/MA20均线
- **成交量图**：红涨绿跌柱状图，与K线图联动
- **统计面板**：最新价、涨跌幅、年内最高/最低价、平均成交量

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `512010_医药ETF易方达_面板.html` | 交互式HTML仪表板（可直接浏览器打开） |
| `512010_医药ETF易方达.csv` | 原始交易数据 |
| `fetch_data.py` | 数据获取脚本 |
| `build_html.py` | 数据注入HTML脚本 |
| `dashboard.html` | HTML模板 |
